# [Paid Memberships Pro - Constant Contact Add On](https://www.paidmembershipspro.com/add-ons/pmpro-constant-contact/) #

![License](https://img.shields.io/badge/license-GPL--2.0%2B-red.svg?style=flat-square)

### Welcome to the Constant Contact Add On GitHub Repository
Sync your PMPro members to a Constant Contact list with tags assigned per membership level.

For more information please visit [www.paidmembershipspro.com/add-ons/pmpro-constant-contact/](https://www.paidmembershipspro.com/add-ons/pmpro-constant-contact/)

## Installation ##
For detailed installation steps, visit the [documentation](https://www.paidmembershipspro.com/add-ons/pmpro-constant-contact/) page.

1. Download the current development ZIP file directly: `https://github.com/strangerstudios/pmpro-constant-contact/archive/master.zip`

**Please ensure that once installing this version of the plugin to remove `-master` from the plugin's folder name.**

## Contributing to the Constant Contact Add On ##
We encourage and welcome any contribution to the Constant Contact Add On. Please read the [guidelines for contributing](https://github.com/strangerstudios/paid-memberships-pro/blob/dev/.github/CONTRIBUTING.md) to this repository.

There are various **ways to the help development** of the Constant Contact Add On:

1. Report [bugs/issues](https://github.com/strangerstudios/pmpro-constant-contact/issues/new) on GitHub.
2. Work on any issues by submitting a Pull Request.

Here are some ways for **non-developers to contribute** to the Constant Contact Add On:

1. Translate the Constant Contact Add On into your own [language](https://www.paidmembershipspro.com/paid-memberships-pro-in-your-language/).
2. [Purchase a paid membership](https://paidmembershipspro.com/pricing) to help fund ongoing development and bug fixes.
3. Leave an honest review for [Paid Memberships Pro - Constant Contact Add On](https://www.paidmembershipspro.com/submit-testimonial/).
